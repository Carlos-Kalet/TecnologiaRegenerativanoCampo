📋 Sobre o Projeto
Este projeto é uma plataforma web completa e interativa sobre Tecnologia Regenerativa no Campo, desenvolvida para demonstrar como a agricultura pode ser produtiva e ambientalmente responsável simultaneamente.

O site aborda:

🌱 Bioinsumos e Microbioma do Solo

🛰️ Agricultura de Precisão Regenerativa

🌳 Sistemas Agroflorestais (ILPF)

⚡ Bioenergia e Economia Circular

🎯 Objetivo
Demonstrar que é possível alcançar alta produtividade agrícola enquanto se restaura o meio ambiente, sequestra carbono e gera rentabilidade para o produtor rural.

🧮 Calculadora de Carbono
A calculadora utiliza fórmulas baseadas em dados científicos:

javascript
// Fórmula de sequestro de carbono
carbonoAnual = area × fatorSequestro × tecnologiaMultiplicador

// Onde:
// - fatorSequestro: 2.5 (convencional), 3.8 (pastagem), 1.2 (integrado)
// - tecnologiaMultiplicador: 1.3 (bioinsumos), 1.8 (ILPF), 2.2 (SAF)

// Valor dos créditos (US$ 25/ton × cotação R$ 5,70)
valorCredito = carbonoAnual × 25 × 5.7

🎯 Prompts Utilizados no Desenvolvimento
Todo o projeto foi desenvolvido através de uma conversa com IA. Abaixo estão os prompts utilizados:

Prompt 1 - Solicitação Inicial
"imagine que você é um programador e está buscando uma solução para 'Agro forte, futuro sustentável: equilibrio entre produção e meio ambiente', e o tema escolhido foi a Tecnologia Regenerativa no Campo, atravez de um site com 3 partes diferentes e separadas : html, css e javascript. Quero que o site n precise de midia interna, e tudo seja possivel de carregar atravez só da internet."

Resposta: Site base com hero section, 4 cards de tecnologias, gráfico simples, formulário de contato e estatísticas animadas.

Resposta: Este README completo.
